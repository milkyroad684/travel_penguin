# 旅するペンギン（travel_penguin）

小学生向けに、日本の47都道府県の **県名と県庁所在地** を覚える Flutter 製の学習ゲームです。リュックを背負ったペンギンと一緒に、日本中を旅しながら覚えましょう。

## 遊び方・モード

- **タイムアタックモード** … 60秒間で、県名カードと県庁所在地カードのペアをできるだけ多くそろえます。連続で正解するとコンボでスコアが伸びます。
- **地方べつモード・全国制覇マップ** … 7つの地方ごとに出題。時間制限はありません。**一度も間違えずにクリア**すると、日本列島マップのその地方が色でぬられ、全国制覇を目指せます。
- **苦手県の復習** … 間違えた県は記録され、復習モードで優先的に出題されます。

## 主な特徴

- 47都道府県すべてに対応
- コンボ・スコア・経験値・ペンギンのレベルアップ
- 学習履歴を端末にローカル保存（`shared_preferences`）
- ペンギンと日本地図は `CustomPainter` で描画（画像アセット不要）
- 画面幅に応じて2列／2段に切り替わるレスポンシブ対応

## 技術スタック

- Flutter / Dart
- 状態管理：Provider（ChangeNotifier）
- 永続化：shared_preferences
